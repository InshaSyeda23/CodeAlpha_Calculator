# CodeAlpha_Calculator
This project is a simple yet functional Basic Calculator developed using HTML, CSS, and JavaScript. The goal of this project is to demonstrate fundamental front-end web development concepts, including user interface design, event handling, and real-time data processing in the browser.
## Features
- Perform basic arithmetic operations: **Addition (+), Subtraction (−), Multiplication (×), Division (÷)**
- Interactive **calculator buttons and display screen**
- **Real-time input display**
- **Clear (AC) and Delete (DEL) functionality**
- Responsive and modern **UI design**
- Rounded display box and styled buttons
- Keyboard input support

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
Users can click the buttons to input numbers and operations. The calculator evaluates the expression and displays the result instantly.

## Demo
You can run the project by opening the `index.html` file in your browser.
